# Bamazon

A CLI mock storefront using node.js and MySQL!

## Installing

Once you've cloned the repository, you'll need to install the required node packages listed in packages.json. This is done with a simple npm install command.

## Built With
* Javascript
* Node
* Inquirer
* cli-table
* MySQL
